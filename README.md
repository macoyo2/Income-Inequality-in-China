# Income Inequality in China
## Overview of the report

This repo contains R scripts and an RMarkdown file used to analyze 1978 to 2016 Urban and Rural income in china. The dataset is obtained from the [A Survey on Income Inequality in China](https://www.aeaweb.org/articles?id=10.1257/jel.20201495&ArticleSearch%5Bwithin%5D%5Barticletitle%5D=1&ArticleSearch%5Bwithin%5D%5Barticleabstract%5D=1&ArticleSearch%5Bwithin%5D%5Bauthorlast%5D=1&ArticleSearch%5Bq%5D=&JelClass%5Bvalue%5D=0&journal=2&from=j). The primary purpose of this report is to analyze the underlying economical and political reasons that cause the Urban-rural disparity in China. The final report can be viewed [here](https://github.com/macoyo2/Toronto-Daily-Shelter-Occupancy/blob/main/outputs/paper/paper.pdf).

Contents
  * R scrips can be found in /scripts
  * R markdown, references, and the final pdf can be found in /outputs
